# Advent of Code 2021 Solutions

My solutions for Advent of Code 2021.
Running requires recent julia installation (I'm using 1.6).
Run tests with

    julia --project=. -e 'using Pkg; Pkg.test("aoc2021")'

or in the julia REPL with `]test`.

Running with input requires [eggnog](https://github.com/breakthatbass/eggnog) to be installed.

    ./run <day> <part>
